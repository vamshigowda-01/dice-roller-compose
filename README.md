# 🎲 Dice Roller App

A simple Android application built using **Kotlin** and **Jetpack Compose** that simulates rolling a dice.  
This app demonstrates **button interaction, state management, and UI recomposition** in a modern Android UI.

---

## 📱 Features
- Roll a dice by tapping a button
- Random dice face generation (1–6)
- Image updates based on dice value
- Clean and minimal Jetpack Compose UI

---

## 🧠 Concepts Demonstrated
- `Button` and `onClick` interaction
- `remember` and `mutableStateOf` for state
- `when` expression for conditional logic
- Recomposition triggered by state changes
- Image handling using Compose

---

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **IDE:** Android Studio
